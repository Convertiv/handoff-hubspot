## [0.2.0]

### Features
- Refactoring the code to use ES modules and build tests
- Added tests for the CLI
- Support for a query builder for HubDB
    - Defines a new config to enable the query builder
    - Adds a diagnostics panel to the transpiler to show available columns
    - Supports x,y, and multi-series chart data
    - Supports limit and sort
- Defines a new config to allow per component js shipping
    - Even if the global js is enabled, you can still ship built js per component
